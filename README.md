## Usage
```bash
# 1. Clone this repo
~ git clone https://github.com/oussamabouchikhi/tailwindcourse.git
# 2. cd into the project folder
~ cd tailwindcourse
# 3. Install dependencies
~ npm install
# 4. Compile files
~ npm run dev
# 5. Run with live-server
# Now you can visit the website at http://127.0.0.1:5500/
```
